## Source

```bash
https://github.com/jsnln/ParametricGaussRecon
```

## Setup

```bash
conda activate pgr python=3.10
conda activate pgr
./setup.sh
```

## Run

```bash
python run_pgr.py <your-pointcloud-file-path>
```
